# AxiomExchange

A production-inspired electronic exchange simulator built in modern C++20.

The goal of this project is to explore the architecture of modern electronic exchanges by implementing a high-performance matching engine, in-memory limit order book, market data dissemination, persistence, and latency benchmarking while following production software engineering practices.

## Features

- Price-Time Priority Matching
- Limit & Market Orders
- Order Cancellation & Modification
- Multi-Threaded Architecture
- Lock-Free Message Queues
- Memory Pool Allocator
- Market Data Feed
- Risk Engine
- Persistent Journal & Replay
- Performance Benchmarking
- Python Analytics Dashboard
